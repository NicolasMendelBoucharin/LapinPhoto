# Photo Book Website

This project is a simple photo book website that showcases two distinct photo galleries: one for Amsterdam and another for Diverse photos. 

## Project Structure

The project consists of the following files and directories:

- **index.html**: The homepage of the photo book website, featuring links to the Amsterdam and Diverse galleries.
- **amsterdam.html**: A dedicated page for the Amsterdam photo gallery, displaying images related to Amsterdam.
- **diverse.html**: A dedicated page for the Diverse photo gallery, showcasing a variety of images.
- **assets/**: This directory contains all the assets for the website.
  - **css/**: Contains the `styles.css` file for styling the website.
  - **images/**: Contains subdirectories for storing images.
    - **amsterdam/**: Directory for Amsterdam-related images.
    - **diverse/**: Directory for Diverse-related images.
- **README.md**: This documentation file, providing an overview of the project and its structure.

## Usage

To view the photo book website, open the `index.html` file in a web browser. From there, you can navigate to the Amsterdam and Diverse galleries.

Feel free to add more images to the respective directories and update the HTML files to include new content as needed.
